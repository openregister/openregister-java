Endpoints

/records returns the most recent item for every key in a register.

/entries returns the log of all entries appended to a register and the hash of the item attached to each entry, ordered by entry number.

/item/{hash} returns an item is a blob of data. Each item in a register is linked to an entry by a unique hash. The entry describes when the item was added to the register.
